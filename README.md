# 在Linux上安装google chrome

主要包括两个版本的chrome安装包，rpm（Centos可用）和deb（Ubuntu可用）包。
**注意：下面的安装包都是64位版本的**

### 使用方法
首先下载相应版本的安装包，下载完成后在本地安装，Centos使用`yum`安装rpm包，Ubuntu中时候`dpkg`安装deb包。


### Centos版本的Chrome
